PIAIC Courses

Artificial Intelligence

Artificial Intelligence programe of PIAIC takes the students through the Foundation of AI technology via teaching the program in open source languages like Python, Numpy, Pandas, Tensorflow and flask.

Cloud Native Computing

Cloud Native Computing teaches the students open source technology like linux, Docker, Kubernetes, JavaSript, Node.js, React and React Native while introducing the students to development of microservices.

Blockchain Computing

Blockchain is a decentralized, digital ledger that records transactions and maintains a continuously growing list of records, called blocks. Each block contains a timestamp and a cryptographic hash of the previous block, creating a chain of blocks, hence the name blockchain.

Internet of Things

The Internet of Things (IoT) refers to the interconnectivity of physical devices, vehicles, buildings, and other objects that are embedded with sensors, software, and network connectivity. This connectivity allows these objects to collect and exchange data with each other and with humans, creating a network of smart, connected devices.
